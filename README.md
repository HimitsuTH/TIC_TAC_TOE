## Set up

```
# Clone this repository
$ git clone https://github.com/HimitsuTH/TIC_TAC_TOE.git

# Go into the repository
$ cd TIC_TAC_TOE

# Install dependencies
$ npm install

# Run 
$ npm run dev

```
